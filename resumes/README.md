# 📄 Resumes

> **"The document that's always too long — until it needs to be longer."**

## What's Here 📋

`ChrisCarpenter_Resume_2026.md` is a comprehensive master resume containing the full detail of my work history, skills, and experience.

**This is not a document I would send to an employer.** It is intentionally long and exhaustive — a reference document meant to be trimmed and tailored into a targeted resume for specific job applications.

## How to Use 🎯

Drop `ChrisCarpenter_Resume_2026.md` into an AI conversation along with the job posting you're targeting. Ask it to tailor a focused, one-to-two-page version that highlights the most relevant experience. The master file has everything — the AI picks what fits.

## Maintenance 🔄

Update the master resume whenever roles, skills, or accomplishments change. Better to keep it current than scramble when an opportunity shows up.
